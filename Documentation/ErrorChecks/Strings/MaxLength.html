MaxLengthÂ¶

Checks if the value of a field has less than the configured length

Example Code:

validators {
  DefaultValidator {
    model = DefaultValidatorModel
    config {
      fields {
        post-code.errorChecks {
          maxLength {
            model = MaxLengthModel
            maxLength = 7
          }
        }
      }
    }
  }
}

Properties

maxLength

Sets the max string length a field value can be.

Mandatory

False

Data Type

Integer

Default

0
