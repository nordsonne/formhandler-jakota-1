Pre ProcessorsÂ¶

You can enter as many Pre Processors as you like. Each entry requires a model name of the PreProcessor. Optionally you can enter specific configuration for the PreProcessor in the config section. The Pre Processors are only called the first time the form is shown.

TypoScript Path

plugin.tx_formhandler_form.settings.predefinedForms.[x].preProcessors
SetSelectOptionsPreProcessor

Sets a named Options array for select boxes.
